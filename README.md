# Realm of Fates — Play (deploy mirror)

로그라이크 카드 오토배틀러 **Realm of Fates** 의 정적 배포 미러입니다.
개발 저장소(private)의 런타임 파일(`index.html` + `css/` + `js/` + `img/` + `snd/`)만 담은 자동 생성 미러로, **GitHub Pages 배포 전용**입니다.

## 플레이

- 🔗 https://chahongpil.github.io/realm-of-fates-play/

## 참고

- 이 미러는 **오프라인(localStorage) 모드**로 동작합니다 — 클라우드 로그인/세이브(Supabase)는 비활성.
- 소스·기획 문서·에셋 원본은 비공개 개발 저장소에서 관리됩니다.
- 외부 의존: GSAP / supabase-js (CDN).
